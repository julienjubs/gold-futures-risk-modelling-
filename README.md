# gold-futures-risk-modelling
One-day-ahead VaR and ES forecasting on 10 years of gold futures data (2016–2026).
Built in R using GARCH(1,1) with Student-t innovations and three forecasting methods:
Dynamic Historical Simulation, EVT/GPD, and parametric Student-t.
Backtested over a 1000-day rolling window using Kupiec and Christoffersen tests.

**Course:** MATH 510 — Quantitative Risk Management, McGill University
